# REE PINN v9 — Climate-Aware Multi-Profile Integration

## 核心思路

把所有剖面的数据整合在一起训练，用气候条件作为额外输入。网络自己学习"深度 + 气候 → REE" 的隐式物理关系。

## vs v8 的核心区别

| | v8 | v9 |
|--|----|----|
| 训练方式 | 每个剖面独立训练 | 所有剖面一起训练 |
| 输入维度 | 1D (仅归一化深度) | **4D (深度 + T + P + R)** |
| 归一化 | 每剖面独立 C_scale | 全局归一化 |
| 跨剖面迁移 | ❌ | ✅ |

## 输入变量

| 变量 | 说明 | 归一化 |
|------|------|--------|
| z_norm | 归一化深度，z / 33.0m | [0, 1] |
| T_norm | 年均温，(T_K - 290.62) / 3.78 | ~[-1, +1] |
| P_norm | 年降水，(P_mm - 1604.84) / 710.27 | ~[-1, +1] |
| R_norm | 径流量，(R_m - 0.521) / 0.231 | ~[-1, +1] |

## 网络架构

```
[z_norm, T_norm, P_norm, R_norm]
  → 4D SIREN (4-64-64-64-64-64-1)
  → ReLU6() × 4000
  → TotalREE (ppm)
```

- SIREN: 5 个隐藏层，每层 64 神经元
- ReLU6 激活：限制输出在 [0, 4000] ppm 范围
- 总参数量: ~21,000

## 损失函数

```
L_total = 100 × L_data + 10 × L_bc
```

- **L_data**: MSE(C_pred, C_obs)
- **L_bc**: C(z=0, T, P, R) = C_atm，C_atm = 所有 0~1m 表层样本均值
